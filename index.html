The Turing Machine

A machine that could solve any math equation given an infinitely long tape with boxes in it that either said 1 and 0 by overwriting values on different positions on the tape given the current value the reader was over
It was the first description of a machine that could be programmed. Modern computers today are all Turing machines that toggle on and off 1's and 0's in a sequence. For this, Turing is known as the father of the modern computer

Code Breaking

During World War II, Alan Turing was tasked by the British to crack Germany's Enigma Code. In Naval warfare, the Germans were sending encrypted messages that contained sensitive combat information using the Enigma Machine. The Enigma Code was a nearly unbreakable cipher that outputted a different character given the same input with 3 knobs that can alter the nature of the encryption.
His work shortened the war by years and it's estimated he saved 15+ million lives.

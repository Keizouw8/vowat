The Turing Machine

A machine that could solve any math equation given an infinitely long tape with boxes in it that either said 1 and 0 by overwriting values on different positions on the tape given the current value the reader was over
It was the first description of a machine that could be programmed. Modern computers today are all Turing machines that toggle on and off 1's and 0's in a sequence. For this, Turing is known as the father of the modern computer
